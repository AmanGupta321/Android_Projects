# Basic Calculator

A simple Android calculator app that performs basic arithmetic operations.

## Features

- Addition, subtraction, multiplication, division
- Clear (AC) and backspace functionality
- Supports decimal numbers
- Vibrate feedback

## Tech Stack

- **Language:** Kotlin
- **Framework:** Android SDK
- **Math Library:** MathParser.org-mXparser
- **UI:** Material Design

## Versions Used

- **Android Studio:** Panda 1 2025 3.1
- **Gradle:** 8.5
- **Gradle Plugin:** 8.2.0
- **Kotlin:** 1.9.20
- **Compile SDK:** 33
- **Target SDK:** 33
- **Minimum SDK:** 21
- **Java Version:** 1.8
- **Dependencies:**
  - Core KTX: 1.7.0
  - AppCompat: 1.6.1
  - Material Components: 1.8.0
  - ConstraintLayout: 2.1.4
  - MathParser.org-mXparser: 4.4.2
  - JUnit: 4.13.2
  - AndroidX Test JUnit: 1.1.5
  - Espresso: 3.5.1

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/AmanGupta321/Android_Projects.git
   ```
2. Navigate to the "Basic Calculator" folder.
3. Open the project in Android Studio.
4. Sync Gradle files.
5. Run the app on an emulator or device.
